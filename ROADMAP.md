# Roadmap

## Current State

TheSecondBrain is a **beta** terminal knowledge vault focused on reliability for internal/team use first, with open-source polish improving over time.

## Current Priorities

1. Correctness and path consistency
2. Regression protection through tests and CI
3. Documentation accuracy
4. Release/install discipline
5. Better contributor trust signals

## In Scope for the Near Term

- wiki path consistency and vault safety
- better verification and health checks
- release tags and downloadable binaries
- clearer troubleshooting and backup guidance
- contributor/security/community repo basics

## Explicitly Not Done Yet

- PDF ingestion
- full migration tooling for broken vault layouts
- enterprise-grade multi-user workflows
- cloud sync or hosted service features

## Future Candidates

- PDF text extraction
- richer import/export flows
- stronger vault repair tooling
- deeper linting and contradiction analysis
