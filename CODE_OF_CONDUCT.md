# Code of Conduct

## Our Standard

Be respectful, direct, and constructive.

Examples of expected behavior:

- assume good intent
- discuss code and behavior, not people
- provide reproducible bug reports
- accept correction without escalation
- keep technical disagreement factual

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language
- doxxing or sharing private information
- aggressive, abusive, or intentionally disruptive conduct

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.
